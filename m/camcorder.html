Native Video Recording

Access the native video recording features with a simple line of JavaScript...

var options = {postURL: myPostURL};
var callback = function(event){...};
function launchVideoCamera(){
    bridgeit.camcorder('myId', 
        callback, options);  
}

<a type="button"
    onclick="launchVideoCamera();">
    Record a video</a>
