Downloads

No downloads required!

You can start using BridgeIt without downloading anything. The BridgeIt script is hosted at:
http://api.bridgeit.mobi/bridgeit/bridgeit.js

Simply add it to your page and you are off and running. The native BridgeIt app will be installed automatically when your application requires it.

Getting the Script
On GitHub: https://github.com/bridgeit/bridgeit.js

Hosted versions:
http://api.bridgeit.mobi/bridgeit/v1/bridgeit.js

Getting the Native Apps:
The BridgeIt native apps will be installed automatically when needed, or you can install one yourself.
