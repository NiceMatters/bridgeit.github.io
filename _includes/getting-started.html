Getting Started with BridgeIt

  1. Include the BridgeIt JavaScript in your page {% highlight html %} {% endhighlight %}
  2. Attach a BridgeIt call/callback to some action element on your page. For instance, a button to retrieve a contact from the address book... {% highlight javascript %} bridgeit.fetchContact('myId', callback); {% endhighlight %}
  3. Access the page from your mobile browser. If BridgeIt is not already installed, you will be prompted to do so. Your application is now BridgeIt enabled.
